Notes
-----

.. note::
   - Tested against NX-OS 8.4(1)



Examples
--------

.. code-block:: yaml

    - name: Test that device alias module works
      cisco.nxos.nxos_devicealias:
        da:
        - name: test1_add
          pwwn: 56:2:22:11:22:88:11:67
        - name: test2_add
          pwwn: 65:22:22:11:22:22:11:d
        - name: dev1
          remove: true
        - name: dev2
          remove: true
        distribute: true
        mode: enhanced
        rename:
        - new_name: bcd
          old_name: abc
        - new_name: bcd1
          old_name: abc1
