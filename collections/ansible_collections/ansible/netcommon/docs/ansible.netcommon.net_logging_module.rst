Notes
-----

.. note::
   - This module is supported on ``ansible_network_os`` network platforms. See the :ref:`Network Platform Options <platform_options>` for details.



Examples
--------

.. code-block:: yaml

    - name: configure console logging
      ansible.netcommon.net_logging:
        dest: console
        facility: any
        level: critical

    - name: remove console logging configuration
      ansible.netcommon.net_logging:
        dest: console
        state: absent

    - name: configure host logging
      ansible.netcommon.net_logging:
        dest: host
        name: 192.0.2.1
        facility: kernel
        level: critical

    - name: Configure file logging using aggregate
      ansible.netcommon.net_logging:
        dest: file
        aggregate:
        - name: test-1
          facility: pfe
          level: critical
        - name: test-2
          facility: kernel
          level: emergency
    - name: Delete file logging using aggregate
      ansible.netcommon.net_logging:
        dest: file
        aggregate:
        - name: test-1
          facility: pfe
          level: critical
        - name: test-2
          facility: kernel
          level: emergency
        state: absent
